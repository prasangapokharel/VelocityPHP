# ⚡ VelocityPhp Framework

Ultra-fast, production-ready PHP framework with modern architecture.

[![GitHub](https://img.shields.io/badge/GitHub-VelocityPHP-blue)](https://github.com/prasangapokharel/VelocityPHP)

## Quick Start

```bash
# 1. Start the server
php start.php

# 2. Open in browser
http://localhost:8000
```

## Features

- ✅ **Ultra-Fast**: Optimized performance with caching
- ✅ **Modern MVC**: Clean separation of concerns
- ✅ **Zero Refresh**: AJAX-powered navigation
- ✅ **Secure**: Built-in security features
- ✅ **Beautiful UI**: Modern design system
- ✅ **Shared Hosting Ready**: 100% compatible with shared hosting
- ✅ **Production Ready**: Fully optimized and tested

## Structure

```
├── public/          # Web root
├── src/
│   ├── config/      # Configuration
│   ├── controllers/ # Controllers
│   ├── models/      # Models
│   ├── views/       # Views
│   └── utils/       # Utilities
├── database/        # Database schemas
├── logs/            # Application logs
└── start.php        # Quick start script
```
